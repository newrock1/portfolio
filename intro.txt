<h2>WEB SITE概要</h2>

<h3>1.  本WEBサイト設立目的</h3>
<ul>
    <li>フリーランスや副業の動画編集者とクライアントをマッチングさせるサイト（動画編集に特化）。</li>
    <li>動画編集者（初心者）対象のスクール講座。</li>
</ul>

<h3>2.  ユーザーが求めていると思われる情報</h3>
<ul>
    <li>動画編集者：動画編集中級者以上対象の高単価の動画編集案件の受注。</li>
    <li>クライアント：中級以上の動画編集者。</li>
</ul>

<h3>3.  ユーザーに当該サイトで何をして欲しいのか</h3>
<ul>
    <li>動画編集者（中、上級者）とクライアントをマッチングさせたい。</li>
    <li>動画編集初心者を動画編集スクールに入校させたい。</li>
</ul>

<h3>4.  メインゴール</h3>
<ul>
    <li>動画編集者とクライアントを一組でも多くマッチングさせ、仲介手数料の売上を伸ばす。</li>
    <li>動画編集希望者（初心者）をスクールに入校させ、売上を伸ばす。</li>
</ul>

<h3>5.  サブゴール</h3>
<ul>
    <li>双方に会員登録をしてもらう。</li>
    <li>当該サイトのサービスの内容を知ってもらう。</li>
    <li>サイトを見てもらう。</li>
    <li>サービスをSNSで広めてもらう。</li>
    <li>レビューを書いてもらう。</li>
    <li>スクールに入校してもらう。</li>
</ul>

<h3>6.  ターゲットユーザー</h3>
<ul>
<li>動画編集者（既にある程度の収益をあげている中、上級者）。</li>
<li>中、上級者以上の動画編集者を求めているクライアント企業。</li>
<li>スクール入校希望の動画編集者（初心者）。</li>
</ul>

<h3>7.  本サイト制作にあたり工夫した点</h3>
<ul>
    <li>ビジネスサイトとして自分が動画編集者になった場合に登録したいと思えるサイトを意識して作成。
    </li>
    <li>極力奇をてらったデザインにならないよう注意しながらも、自分のオリジナリティーを消さずにビジネスサイトを意識して作成。
    </li>
    <li>デザインを奇抜ながらも、メニューボタン等できる限り分かりやすく実装。</li>
</ul>

<h3>8. 制作期間</h3>
<ul>
    <li>約２週間。</li>
</ul>

<h3>9. 使用可能なソフト・エディタ―</h3>
<ul>
    <li>Adobe Illustrator 2022 2023</li>
    <li>Adobe Photoshop 2022 2023</li>
    <li>Adobe Dreamweaver 2021 2022</li>
    <li>Microsoft Visual Studio Code</li>
    <li>WordPress 最新版</li>
    <li>Adobe Premiere pro 2023(独学)</li>
</ul>

<h3>10. 主に自分ができること</h3>
<ul>
    <li>MOSのWord・Excel・PowerPoint・Outlook資格取得済み及び書類作成。</li>
    <li>Photoshopを使用した画像の加工。</li>
    <li>illustratorを使用したデザイン。</li>
    <li>HTML、CSS、JavaScrip、jQueryの基礎知識があり、コーディング実装可能。</li>
    <li>Web解析・SEO対策。</li>
</ul>
